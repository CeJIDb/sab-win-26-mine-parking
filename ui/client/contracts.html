Управление договорами

Мои договоры

Список договоров: активные, на согласовании, завершённые. Для каждого — срок, тариф, зона/места, статус оплаты. Действия: оформить новый договор, продлить, запросить расторжение, скачать документы.

Действия по договору

Заполнение данных для договора, подписание, оплата. Уведомления о приближении срока оплаты и окончании действия. Связь с бронированием: долгосрочная бронь оформляется как договор.

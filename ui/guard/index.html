Интерфейс КПП

Проверка доступа

Ввод номера ТС для проверки допуска. Отображение статуса (разрешён/отказ) и причины отказа. Резервный способ идентификации при сбое автоматики.

Исключения и офлайн-клиенты

Обработка нестандартных ситуаций: офлайн-клиент (оплата на выезде через терминал/QR), ручное открытие шлагбаума при поломке, фиксация отказа от оплаты, ручная фиксация выезда при сбое системы учёта.

Текущие сессии

Просмотр текущих парковочных сессий, занятость по секторам. Категория ТС на мониторе при въезде. Проверка данных о ТС на конкретном месте.

Вход только через авторизацию рабочего интерфейса. Учётные записи охранников создаются в разделе «Управление профилями админов».

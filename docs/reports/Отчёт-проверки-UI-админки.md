# Отчёт проверки UI админ-панели (ui/admin)

**Дата:** 2026-02-28  
**Основа:** инструкция субагента **frontend-ux-ui** (`.cursor/agents/frontend-ux-ui.md`).  
**Область проверки:** `ui/admin`, шаблоны `layouts/admin.njk`, `partials/admin-*`, `pages/admin-*.njk`, общие стили `ui/scss/`.

**Актуализация (правки внесены):** по рекомендациям отчёта выполнены: добавлен `scope="col"` во все таблицы админки; блок `header_subtitle` выведен в layout и в partial шапки; в модалках с submit добавлен блок для вывода ошибки (`.wf-error`); в layout админки добавлены футер с переключателем «для компьютера / для телефона», бургер-меню навигации и скрипты (переключение вида, открытие/закрытие навигации); в `ui/scss/_lk.scss` добавлены стили мобильной навигации для `.wf-nav-wrap--admin`; пункт «Бронирования» добавлен в карту навигации. После изменений необходимо выполнить сборку: `npm run build:css`, `node scripts/build-templates.mjs`.

---

## 1. Цель и критерии

Проверка соответствия интерфейса ЛК администратора заявленным в агенте правилам:

- **Доступность:** семантика разметки, skip-link, `aria-*`, связка `label`/`input`, контраст и фокус.
- **Навигация:** соответствие карте навигации и layout’ам.
- **Обратная связь:** состояния кнопок/форм, сообщения об успехе/ошибке.
- **Консистентность:** переиспользование layout’ов, partials, SCSS.
- **Технологический стек:** только Nunjucks и SCSS, сборка через скрипты, без правок `wireframe.css`.
- **Мобильная адаптация:** при необходимости — переключение desktop/mobile и адаптивные сетки.

## 3. Дополнительные замечания

- **Ссылка «Создать профиль»** на `admin/clients.njk` ведёт на `../site/register.html`. Если подразумевается приглашение клиента к самостоятельной регистрации — ок; если создание профиля администратором — целевая страница должна быть иной (например, форма в админке или отдельная страница).
- **Закрытие модалок:** в `admin.njk` есть скрипт, который по клику на `.wf-modal__close[href="#main-content"]` выполняет `history.back()`. Поведение согласовано с ЛК и с разметкой модалок.

## Разделы

- [Индекс артефактов](artifacts/readme.md) — собирает аналитические артефакты, AS-IS/TO-BE материалы, карты и use case.
- [Индекс спецификаций](specs/readme.md) — объединяет функциональные и нефункциональные требования, а также ограничения.
- [Индекс архитектуры](architecture/readme.md) — ведет к ADR, DDD, C4, архитектуре данных и ERD.
- [Процессная документация](process/readme.md) — описывает правила участия, шаблоны, контроль качества и релизные чеклисты.
- [Индекс интервью](interviews/readme.md) — собирает планы интервью, протоколы встреч и источники решений.
- [Индекс Demo Days](demo-days/readme.md) — объединяет сценарии демо, заметки и презентационные материалы по этапам.

## Вспомогательные инструменты

- [Скрипты для документации](../scripts/docs/readme.md) — вспомогательные утилиты для проверки и подготовки материалов.
